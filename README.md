# Audio Transcription using Whisper

Pre-trained ASR (Automatic Speech Recognition) pipeline that transcribes audio files into text using OpenAI's Whisper model.

## Problem Statement
How can we leverage pre-trained ASR systems to convert spoken audio files into clean, accurate text transcriptions?

## Features
- Multi-format audio support (mp3, wav, ogg, m4a)
- Automatic language detection
- Segment-level timestamps with confidence scores
- Batch processing of multiple audio files

## Tech Stack
Python, OpenAI Whisper, PyTorch, FFmpeg, Google Colab

## How to Run
1. Open `Untitled7.ipynb` in Google Colab
2. Run all cells in order
3. Upload audio files when prompted
4. View transcription output with timestamps and confidence scores
